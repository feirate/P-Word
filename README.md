# P-Word 英语口语练习小程序

> 一款专注英语口语练习的微信小程序，帮助用户利用碎片时间提升口语输出能力
> 
> **AI 全栈开发项目** - 使用AI助手完成从设计到开发的全流程

## 🎯 项目特色

- 🎙️ **极简录音**：5秒快速启动，一键录音练习
- 📊 **实时反馈**：波形可视化，即时回放体验
- 📚 **智能语料**：分级语料库 + 自定义导入
- 📈 **进度追踪**：热力图日历，见证每日进步
- 🔊 **语音评分**：本地化评分系统（可选）

## 🏗️ 技术架构

- **前端**：微信小程序原生开发
- **云服务**：微信云开发（免费5GB）
- **音频处理**：wx.getRecorderManager() + Canvas波形
- **数据存储**：本地缓存 + 云端同步
